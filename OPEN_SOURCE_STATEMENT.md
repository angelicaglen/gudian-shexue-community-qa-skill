# 古典射学社群问答 Skill 开源声明

古典射学社群问答 Skill 是一个面向传统射艺学习、资料检索和社群问答的开源项目。它通过公开的 skill 代码、远程知识库 API 和持续更新的古典射学资料整理流程，为社群成员提供可复核、有边界的问答能力。

## 核心承诺

1. Skill 源代码开源，遵循 MIT License，可查看、复制、修改、自部署和二次开发。
2. 古典射学知识库的基础检索 API 长期免费开放，用于传统射艺学习、公开资料查阅、社群问答和合规研究。
3. 公共知识库不设置隐藏收费、内容锁闭或付费解锁门槛。
4. 项目鼓励社群成员提交史料、考据、技术术语修正、视频转录纠错和使用反馈，共同提升知识库质量。
5. 免费测试版会收集用户问题、模型回答、引用来源、反馈和必要的运行信息，用于改进知识库、检索质量和回答质量。

## 代码、知识库与 API 的边界

本项目分为三层：

- Skill 程序代码：使用 MIT License 开源。
- 知识库整理内容：用于免费检索、学习和社群问答；其中由项目自行整理形成的文本可在保留来源、署名和上下文边界的前提下用于非商业学习与研究。第三方原始材料、公开视频、平台内容、古籍影印、他人文章和投稿材料仍受各自权利人、来源平台或提交约定约束。
- 公共 API：用于连接 skill 和知识库检索服务。基础调用长期免费，但项目可为反滥用、安全、合规、维护和服务稳定设置限流、版本升级、异常调用拦截或临时维护。

开源代码不等于放弃知识库资料来源的署名、引用和合规边界；免费 API 不等于无限制调用、商业转售或绕过数据政策。

## 数据收集提示

使用免费测试版公共 API，即表示你理解并同意：本项目会收集你的问题、模型回答、引用来源、反馈、skill 版本、客户端自报模型信息和必要的质量标记，用于改进知识库与问答服务。

本项目不主动采集手机号、微信、真实姓名、身份证号、住址、精确位置等个人身份信息。请不要在提问中输入个人隐私或他人隐私。若用户自行输入隐私内容，可能随问题文本一并提交；项目会尽量进行脱敏、清理和最小化保存。

## 使用限制

本项目仅用于传统射艺文化学习、公开资料整理、知识库问答、社群讨论和合规研究。禁止用于：

- 违法、危险、伤害他人或规避监管的用途；
- 侵犯隐私、骚扰、跟踪、人肉搜索或收集他人身份信息；
- 冒充官方机构、伪造来源、篡改引用或误导公众；
- 大规模爬取、压测、转售公共 API 或绕过反滥用限制；
- 将未经审核的回答包装成正式训练规范、安全规范、医疗建议或法律意见。

## 免责声明

本项目提供的是资料检索和问答辅助，不替代线下教练、安全员、活动组织方、专业研究者或法律顾问的判断。涉及弓具使用、训练动作、活动安全和历史考据时，应保留来源、时间、上下文和不确定性。

公共 API 按现状提供。项目会尽力维护服务可用性和资料质量，但不承诺持续无中断、无错误、适合任何特定目的或满足生产级 SLA。

## 社群共建

欢迎提交：

- 古射典籍原文、校勘资料和可靠出处；
- 传统射法、器材、礼仪、比赛和活动资料；
- 视频转录纠错、术语修正和时间点复核；
- 错误回答、缺失材料和引用问题反馈；
- 对安全边界、争议表达和社群规范的改进建议。

提交内容进入知识库前需要经过整理、脱敏、来源标注和人工审核。项目保留拒收、延后收录、修改格式、标注不确定性和撤回不合适内容的权利。
