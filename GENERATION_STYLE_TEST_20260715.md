# 生成式回答风格测试

测试时间：2026-07-15

本次调整目标：让 skill 从“证据摘要式回答”转向“回答优先，证据托底”的证据增强生成模式。

## 样例 1：为什么参加古典射学活动

问题：

```text
我为什么要参加古典射学活动？不要像审计报告，像人话回答。
```

新版回答样例：

```text
如果你只是想体验射箭，普通射箭馆可能更直接；但如果你想接触的是一整套围绕传统弓箭展开的实践，古典射学活动会更适合。它不只是“射几箭”，还包括器物怎么用、现场规则怎么守、人与人怎么在安全和秩序里交流。

所以参加它的价值，不是去比谁更强，而是进入一个有人组织、有人校正、有共同规范的场域。你可以学习，也可以反馈规则；但前提是认同活动规范，尊重现场安全安排。

依据：《古典射学联盟章程》（KB-20260713-0002，【P011】）说明会员需认同联盟宗旨、核心原则与活动规范；《2026年全国性古典射学活动意向提交说明》（KB-20260713-0003，【P007】【P011】）要求活动方案说明流程、安全方案、风险控制，并落实统一规范。更深的礼射和文化解释还需要补充文化/考据材料。
```

要点：

- 没有开头说“文化、考据缺失”。
- 先回答用户的动机问题。
- 末尾只轻提示文化/考据仍需补充。

## 样例 2：现在选弓应该按什么标准

问题：

```text
现在选弓应该按什么标准？
```

新版回答样例：

```text
如果是为了参加联盟相关活动，选弓先不要问“哪把更厉害”，先问“能不能过活动规则”和“自己能不能安全稳定地用”。

按现有知识库，活动检录层面至少看四件事：弓形是否属于允许范围，弓把是否符合传统形制，配箭重量是否符合拉力要求，箭矢的 FOC 和长度是否在规则内。具体到远射交流会规则，允许长稍弓、小稍弓、通用弓或大稍弓；弓把不能是纯推弓把或异形弓把。50 磅及以上按 0.33-0.5 克/磅配箭，50 磅以下单箭不超过 25 克；FOC 不超过 10%，箭全长不超过本人臂展一半。

但这只是活动使用/检录规则，不等于完整的器材学结论。当前知识库的器材、数据、考据材料还不足，所以不能进一步判断哪种弓“性能更好”或“历史上更正统”。

依据：《2026年古典射学远射交流会暨记录会章程》（KB-20260713-0001，【P013】【P014】【P015】）。
```

要点：

- 对强规则问题保留明确边界。
- 能给的规则直接给，不先说“知识库不足”。
- 将“活动检录规则”和“器材学定论”分开。
