# 古典射学知识库公共 API 使用协议

本协议适用于古典射学社群问答 Skill 默认连接的公共知识库 API，包括腾讯云 CloudBase 主 API、Cloudflare 备用 API，以及后续由项目方公开说明的同类接口。

继续调用公共 API，即表示你已阅读、理解并同意本协议。

## 1. 服务内容

公共 API 提供古典射学知识库的基础检索、路由检索、RAG 检索、版本检查、session 启动和问答反馈上传能力。API 返回内容主要用于辅助 skill 生成自然语言回答，不应被直接视为正式规范、训练处方或权威结论。

## 2. 免费开放承诺

本项目承诺：古典射学知识库的基础检索 API 长期免费开放，用于传统射艺学习、公开资料查阅、社群问答和合规研究。

为保证公共服务可持续运行，项目方保留以下权利：

- 设置调用频率、并发量、请求体大小和超时时间限制；
- 拦截异常、恶意、自动化滥用或疑似攻击请求；
- 要求旧版 skill 升级后继续调用；
- 在维护、迁移、故障、安全事件或成本异常时临时暂停服务；
- 调整接口协议、字段结构、认证机制和备用服务地址，并通过版本文件或更新提示说明。

免费开放不代表无限制调用，不代表允许转售公共 API，也不代表承诺生产级 SLA。

## 3. 用户数据与日志

公共 API 免费测试版会收集以下信息：

- 用户问题；
- 模型最终回答；
- 被引用的知识库来源、链接、段落号和时间点；
- interaction_id、session_id、skill 版本、客户端自报模型信息；
- 质量标记、错误反馈、调用时间和必要的服务诊断信息。

这些数据用于：

- 改进知识库内容；
- 发现缺失材料和错别字；
- 优化检索、路由和回答质量；
- 审计高风险回答；
- 排查服务故障和滥用行为。

本项目不主动采集手机号、微信、真实姓名、身份证号、住址、精确位置等个人身份信息。请不要在问题中输入个人隐私或他人隐私。若用户自行输入隐私内容，可能随问题文本一并提交；项目会尽量进行脱敏、最小化保存和后续清理。

## 4. 禁止用途

不得使用公共 API：

- 从事违法、危险、伤害他人或规避监管的行为；
- 收集、推断、暴露或传播个人隐私；
- 冒充项目方、社群、活动组织方或官方机构；
- 伪造来源、篡改引用、误导公众或恶意断章取义；
- 大规模爬取、压测、扫描、攻击或绕过限流；
- 转售、出租、封装售卖公共 API；
- 将未经审核的回答包装为正式训练规范、安全规范、医学建议、法律意见或商业承诺；
- 上传恶意代码、违法内容、侵权材料或与传统射艺学习无关的大量垃圾请求。

## 5. 知识库内容边界

API 返回的知识库材料可能包含项目整理文本、公开视频整理、公开资料摘录、古籍材料、社群章程、活动资料和用户反馈形成的待审核内容。

使用者应保留来源、作者、链接、发布时间、KB 编号、段落号、时间点和不确定性标记。不得把单一视频、单一作者观点、待审核材料或自动转写结果泛化为正式规范。

第三方原始材料仍受各自权利人、来源平台或提交约定约束。公共 API 的免费使用不代表项目方转授第三方原始材料的全部版权。

## 6. 账号、session 与访问控制

公共 API 可要求 skill 调用 `/session/start` 获取 session token。session token 仅用于调用公共 API，不得公开、转售、写入公开日志或提交到代码仓库。

如果用户明确不同意免费测试版的数据收集，skill 不应继续调用公共 API。用户可选择自部署私有 API，并按私有部署策略关闭交互日志上传。

## 7. 服务变更与终止

项目方可以因维护、故障、安全、合规、滥用、成本异常或协议升级暂停、限制或终止部分调用。对于严重滥用、攻击、转售或违法用途，项目方可直接封禁相关调用来源。

项目方会尽量通过 `skill-version.json`、API 更新提示、README、发布说明或社群公告说明重要变更。

## 8. 免责声明

公共 API 按现状提供。项目方不保证服务持续可用、返回内容完全准确、适合任何特定目的或不会中断。

使用者应自行判断回答和来源是否适用于具体场景。涉及训练动作、器材安全、活动组织、法律责任、医疗健康或高风险决策时，应咨询有经验的线下人员或专业人士。

## 9. 协议更新

本协议可能随项目发展更新。继续使用公共 API 视为接受更新后的协议。重大变更会尽量通过版本文件、README 或项目公告说明。
