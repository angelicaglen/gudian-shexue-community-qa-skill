# 资料提交与社群共建说明

感谢你参与古典射学知识库共建。这个项目希望把散落在视频、古籍、活动资料、社群讨论和个人经验中的传统射艺知识，整理成可检索、可复核、有边界的公共学习资料。

## 欢迎提交什么

欢迎提交：

- 古射典籍原文、校勘版本、可靠出处和现代研究资料；
- 传统射法、器材、礼仪、活动规则和社群章程资料；
- 公开视频链接、时间点、标题、作者和发布时间；
- 自动转写错别字、术语误识别和时间点纠错；
- 弓形、箭重、FOC、测速、拉距、扳指形制等数据资料；
- 使用 skill 时遇到的错误回答、缺失材料和引用问题；
- 对安全边界、争议回应、社群规范和对外表达的改进建议。

## 提交时请尽量提供

- 原始来源链接或文件名；
- 作者、发布者或来源机构；
- 发布时间或资料年代；
- 具体页码、段落号、视频时间点；
- 你认为需要修正或收录的内容；
- 你自己的判断和不确定点。

示例：

```text
资料类型：视频转写纠错
原始链接：https://www.bilibili.com/video/...
视频标题：……
作者：……
时间点：05:42-05:49
当前转写：……
建议修正：……
理由：字幕/画面/上下文显示……
```

## 不要提交什么

请不要提交：

- 他人手机号、微信、住址、真实姓名、身份证号等隐私；
- 未经授权的内部群聊原文、私聊截图或收款信息；
- 来源不明、无法复核的断言；
- 与古典射学学习无关的大量材料；
- 违法、危险、攻击性、骚扰或侵犯他人权益的内容；
- 明知侵权、未经授权复制的付费课程、书籍全文或平台受限内容。

## 审核与入库

提交内容不会自动进入正式知识库。项目方会根据情况进行：

- 来源核验；
- 隐私和敏感信息脱敏；
- 术语校订；
- 重复内容合并；
- 转写与摘要纠错；
- 标注审核状态和不确定性；
- 拆分为适合 RAG 检索的知识库条目。

项目方保留拒收、延后收录、拆分、改写、标注不确定性、撤回和删除不合适内容的权利。

## 投稿授权

提交资料即表示你同意项目方在古典射学知识库建设中使用该资料，包括整理、脱敏、改写、标注来源、生成摘要、制作问答材料和纳入知识库候选。

如果你提交的是自己原创内容，请在提交时说明希望如何署名、是否允许公开引用、是否仅供内部参考。

如果你提交的是第三方资料，请尽量提供原始来源。第三方原始材料仍受其权利人和来源平台规则约束。

## 反馈错误回答

如果 skill 回答错了，请尽量提供：

- 你的原问题；
- 错误回答片段；
- 正确资料或反例；
- 你认为应引用的来源；
- 是否涉及安全、隐私或争议风险。

错误反馈会优先用于改进检索、术语词典、知识库清洗和回答边界。
